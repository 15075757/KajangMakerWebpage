# Kajang Maker — Official Website

Official landing page for **Kajang Maker**, a hands-on STEM, AI, and Robotics education provider based in Kajang, Malaysia.

## About

Kajang Maker empowers students through interactive workshops, competitions, school partnerships, and mentorship programs. Founded in 2015, we have been building the problem-solvers of tomorrow across Kajang and the Klang Valley.

## Live Site

[www.kajangmaker.my](https://www.kajangmaker.my)

## What's Inside

- Single-file `index.html` — all styles inline, no build step required
- Animated navbar with mascot character
- Photo gallery from real events and workshops
- Animated CTA section with floating background paths
- Fully responsive, mobile-first design
